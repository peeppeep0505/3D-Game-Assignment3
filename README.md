
# Crystal Collector 3D Game

## Features

- Player movement with keyboard
- Third-person camera that follows the player
- Crystal collecting system
- Enemy movement
- Collision detection
- Win and restart system

## Game Objective

Collect all crystals in the map.

Do not touch the enemies.

- If the player collects all crystals, the player wins.
- If the player touches an enemy, the game restarts.

## Controls

- **W** = Move forward
- **S** = Move backward
- **A** = Turn left
- **D** = Turn right
- **Mouse** = Change camera view
- **Mouse Wheel** = Zoom in / zoom out
- **R** = Restart game after win or lose
- **ESC** = Close the game
